  1. 규칙

색상 테마 관리

기본 사항

  1. TVUT은 기본적으로 다크 모드 테마를 기본으로 하고 있다.
  2. 추후 라이트 모드, 키즈 모드, 시니어 모드 등 다양한 색상 테마가 추가 될 수 있을 것으로 보인다.
  3. 그에 따라 색상 Variable은 테마에 따라 변경 될 수 있도록 구성한다.

색상 정의

  1. 해당 테마 map 내에 CSS var 형태로 색상을 정의한다. 
    $color-map-dark: (
    --color-fg-default: #fff,
    --color-fg-muted: #999,
    --color-fg-subtle: #444,
    --color-fg-on-emphasis: #fff,
);
$color-map-light: (
    --color-fg-default: #000,
    --color-fg-muted: #444,
    --color-fg-subtle: #999,
    --color-fg-on-emphasis: #fff,
);

사용 방법

  1. 미리 정의된 mixin이 색상이 정의된 map을 body의 data-color-theme 속성에 따라 CSS var 형태로 컴파일한다. 
    @mixin spread-map($map: ()) {
    @each $key, $value in $map {
        #{$key}: $value;
    }
}
:root body[data-color-theme=light] {
    @include spread-map($color-map-light);
}
:root body[data-color-theme=dark]{
    @include spread-map($color-map-dark);
}
  2. 따라서 SCSS내에서 색상 사용 시 var(색상명)형태로 색상을 사용해야 한다. 
    body{
    background-color: var(--color-canvas-default);
}
