  1. 규칙

modal : default

기본 사항

  1. 설계서상 별도의 디스크립션이 없을 경우(default)
  2. 디자인상 딤드처리 되어보이지 않음 + 바닥 컨텐츠 컨트롤 불가 + 팝업 컨텐츠 영역 외 클릭 시 팝업 닫기

default type modal

                                <div id="" class="modal">
    <div class="modal__dimmed"></div><!-- 클릭 시 팝업 닫기 기능 추가할 DOM -->
    <div class="modal__container">
        <div class="modal__body">
            <p class="modal__message">TEXT MESSAGE</p>
            CONTENTS DOM
        </div>
        <div class="modal__foot">
            <button type="button" class="btn btn--sm btn--neutralOnOverlay modal__btn">서브버튼</button>     
            <button type="button" class="btn btn--sm btn--emphasisOnOverlay modal__btn">주버튼</button>     
        </div>
    </div>
</div>

modal : 예외처리

기본 사항

  1. 설계서상 팝업 예외처리 디스크립션 있을 경우
  2. 디자인상 딤드처리 되어보임 + 바닥 컨텐츠 컨트롤 불가 + 팝업 컨텐츠 영역 외 클릭 시 팝업 닫히지 않음

dimmed type modal

                                <div id="" class="modal modal--dimmed"><!-- .modal--dimmed 클래스 추가 + .modal__dimmed DOM 없음 -->
    <div class="modal__container">
        <div class="modal__body">
            <p class="modal__message">TEXT MESSAGE</p>
            CONTENTS DOM
        </div>
        <div class="modal__foot">
            <button type="button" class="btn btn--sm btn--neutralOnOverlay modal__btn">취소</button>     
            <button type="button" class="btn btn--sm btn--emphasisOnOverlay modal__btn">뒤로가기</button>     
        </div>
    </div>
</div>
